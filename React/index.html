What is React

React is a javascript library for building user interfaces

React.js is a client-side library. It's about client-side javascript.

The JS code which runs in the browsers of your visitors. Use to manipulate web pages after they were loaded which allows us to build modern, reactive user interfaces for the web. Mobile apps annd desktop apps feel very "reactive": Things happen instantly, you don't wait for new pages too load or actions to start.
Traditionally, in web apps, you click a link and wait for a new page to load. You click a button and wait for some action to complete
